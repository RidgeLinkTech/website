# RidgeLink Technologies Static Website

Static Astro website for the fictional company **RidgeLink Technologies**.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloudflare Pages

Recommended settings:

- Framework preset: Astro
- Build command: `npm run build`
- Build output directory: `dist`
- Production branch: `main`
- Custom domain: `ridgelink.us`

## Purpose

This site is designed for portfolio demos, ServiceNow examples, PowerShell automation examples, cybersecurity scenarios, homelab documentation, and fictional enterprise content.
